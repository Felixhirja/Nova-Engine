  <p class="small">Notes:
    <ul>
      <li>The project's C++ SVG rasterizer (src/SVGSurfaceLoader.cpp) supports a small subset of SVG: shapes (rect/circle/ellipse/polygon/polyline/line/path) using fills.
      </li>
      <li>It does not currently support gradients, filters, strokes, text, &lt;use&gt;/href resolution, or style references stored in &lt;defs&gt;. As a result the in-engine rasterization will miss most stroked/gradient elements from this HUD.</li>
      <li>Recommended quick checks: open this page in a modern browser (Chrome/Edge/Firefox) to see the intended visual. To get an accurate rasterized PNG for the engine, either export a PNG from a vector editor or extend the loader to support strokes/gradients.</li>
    </ul>
  </p>
